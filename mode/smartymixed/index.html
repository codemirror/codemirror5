CodeMirror: Smarty mixed mode

The Smarty mixed mode depends on the XML, JavaScript, and CSS modes.

It takes the following configuration options:

  • leftDelimiter - Smarty's left delimeter character. Default is "{".
  • rightDelimiter - Smarty's right delimeter character. Default is "}".
  • smartyVersion - Smarty version. Possible values are: 2 and 3. Default is 3.

MIME types defined: text/x-smarty (the same as for the Smarty mode, so don't use them both at a time).
