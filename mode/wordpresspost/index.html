WordPress Post Editor mode


The WordPress Post mode depends on the XML, JavaScript, CSS, HTMLmixed, and Shortcode modes. The Shortcode mode is included in the wordpresspost directory.

For use in the WordPress tinyMCE post/page editor in the admin backend. Highlights the WordPress Shorcode Syntax as if were HTML. Also looks for special WordPress <--more--> tags and colors them accordingly.
